FAQ - Perguntas Frequentes

O que é Front-end?
Front-end é a parte visual de um site ou aplicação web, com a qual o usuário interage diretamente.
O que é JavaScript?
JavaScript é uma linguagem de programação que permite criar interatividade e funcionalidades em páginas web.
Como começo a programar?
Comece estudando linguagens como HTML, CSS e JavaScript, que são base para o desenvolvimento web.
